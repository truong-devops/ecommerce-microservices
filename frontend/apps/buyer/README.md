# Buyer app (Expo SDK 54)

React Native + TypeScript buyer application using Expo SDK 54.

## Scripts

- `npm run dev --workspace frontend/apps/buyer`
- `npm run android --workspace frontend/apps/buyer`
- `npm run ios --workspace frontend/apps/buyer`
- `npm run web --workspace frontend/apps/buyer`

## Environment

Set API gateway base URL:

- `EXPO_PUBLIC_API_BASE_URL=http://localhost:8080`
